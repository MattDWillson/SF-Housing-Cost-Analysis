# SF-Housing-Cost-Analysis

Tools: Jupyter Lab, Python, Panel, Plotly, Pandas, hvPlot, Maplotlib, Numpy, Mapbox API

The goal of this dashboard is to provide charts, maps, and interactive visualizations that help customers explore San Fransisco real estate and determine if they want to invest in rental properties. Below are the images included in the dashboard.
